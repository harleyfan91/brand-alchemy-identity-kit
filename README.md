# Brand Alchemy Identity Kit

Monorepo scaffold for the Identity Kit microsite and API.

## Apps

- `apps/web`: React + TypeScript + Vite + Tailwind CSS
- `apps/api`: Node.js + TypeScript + Express

## Quick Start

```bash
npm install
npm run dev:web
npm run dev:api
```

## Environment

Copy `.env.example` values into your provider environment settings (and local `.env` files as needed).
