# WAVE D — Social Casual: `direct_sales`, `lead_gen`, `retention` Tier-1 Depth

Tier-1 depth expansion for the Social Casual surface (Instagram, TikTok, Facebook, Pinterest, Threads) across `direct_sales`, `lead_gen`, and `retention` for all seven industries.

Same methodology as prior waves: new pairs 4–8 added to each existing leaf. Sections marked **NEW SECTION** have no existing pairs; all 8 pairs per tone are provided.

> **Platform register reminders (carry through every pair):**
> - Platform-native mechanics: DM, link in bio, comment, tap, save, drop a DM
> - Fragments are natural — no period needed after a short declarative fragment in caption context
> - Bold: keyword trigger mechanics (DM 'X', Comment 'Y'), fast and decisive
> - Friendly: warm invitation, "we're real people," reply-with-warmth tone
> - Scarcity for `retail_maker`: "get them before they're gone" — not anthropomorphized ("they don't last long" is OK; "they won't stick around" is not)
> - `community`: purpose and belonging register throughout — no transactional efficiency language

## Existing bank fixes — action items before applying Wave D

Two issues in the existing Social Casual base pairs require a Wave A-style fix:

1. **Line 1393 — format anomaly** (`retail_maker > audience_growth > bold > pair 3`):
   ```
   ["Save this and follow. New things every week, everything made by hand."]
   ```
   Parses as single-element array. Recommended fix:
   ```
   ["Save this and follow.", "New things every week, everything made by hand. Get them before they're gone."]
   ```

2. **Line 1227 — register slip** (`community > direct_sales > bold > pair 3`):
   ```
   ["Link in bio to sign up!", "Takes 30 seconds."]
   ```
   Transactional efficiency language is off-register for community. Recommended fix:
   ```
   ["Tap the link and find your place in the work.", "Every kind of contribution matters."]
   ```
